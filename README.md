# Intelligence Hub (MVP) - Streamlit

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Cloud
- Push `app.py` and `requirements.txt` to GitHub
- On Streamlit Cloud, set main file to `app.py`

## Data
- Use the app sidebar:
  - Seed Demo Data
  - Import CSV to replace tables (MVP behavior)
